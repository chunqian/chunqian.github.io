# rnd_next_double

Returns the next random double in the sequence.

## Syntax

```cpp
float rnd_next_double(rnd_t* rnd);
float rnd_next_double(rnd_t& rnd);
```

## Function Parameters

Parameter Name | Description
--- | ---
rnd | The state struct used to generate random numbers.

## Return Value

Returns the next random double in the sequence.
