# batch_outlines

Sets whether or not to use outlines on sprites.

## Syntax

```cpp
void batch_outlines(batch_t* b, bool use_outlines);
```

## Function Parameters

Parameter Name | Description
--- | ---
b | The batch.
use_outlines | True to use outlines for sprites, false otherwise.
