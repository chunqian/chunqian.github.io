# batch_outlines_color

Sets the color for sprite outlines, if used.

## Syntax

```cpp
void batch_outlines_color(batch_t* b, color_t c);
```

## Function Parameters

Parameter Name | Description
--- | ---
b | The batch.
c | Color to use for the outlines.
