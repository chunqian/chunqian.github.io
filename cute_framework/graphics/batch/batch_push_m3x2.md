# batch_push_m3x2

Pushes a matrix 3 x 2 to transform pushed geometry by.

## Syntax

```cpp
void batch_push_m3x2(batch_t* b, m3x2 m);
```

## Function Parameters

Parameter Name | Description
--- | ---
b | The batch.
m | 3x3 matrix to transform geometry with.
