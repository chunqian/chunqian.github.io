# sprite_t::animation_interpolant

Returns a value from 0 to 1 representing how far along the animation has played. 0 means just started, while 1 means finished.

## Syntax

```cpp
float animation_interpolant();
```

## Return Value

Returns a value from 0 to 1 representing how far along the animation has played. 0 means just started, while 1 means finished.
