# sprite_t::is_playing

Returns true if `animation` the is currently playing animation.

## Syntax

```cpp
bool is_playing(const char* animation);
```

## Function Parameters

Parameter Name | Description
--- | ---
animation | Checks to see if this animation is currently playing.

## Return Value

Returns true if `animation` the is currently playing animation.
