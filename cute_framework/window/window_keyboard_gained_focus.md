
# window_keyboard_gained_focus

Returns whether or not the window has gained keyboard focus.

## Syntax

```cpp
bool window_keyboard_gained_focus(app_t* app);
```

## Return Value

Returns true if the window has gained keyboard focus, false otherwise.
