
# window_position

Retrieves the position of the window.

## Syntax

```cpp
void window_position(app_t* app, int* x, int* y);
```

## Function Parameters

Parameter Name | Description
--- | ---
app | The application.
x | The current x position of the window.
y | The current y position of the window.
