# 类型系统

Rust 提供了多种机制，用于改变或定义原生类型和用户定义类型。接下来会讲到：
* 原生类型的[类型转换][Casting]（cast）。
* 指定[字面量][literals]的类型。
* 使用[类型推断][type inference]（type inference）。
* 给类型[取别名][Aliasing]（alias）。
